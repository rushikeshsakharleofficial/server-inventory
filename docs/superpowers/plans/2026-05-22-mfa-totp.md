# MFA (TOTP) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add TOTP-based MFA to ServerInventory — users enroll via /setup page, login becomes two-phase when MFA is active.

**Architecture:** Login endpoint now returns either a full access token (MFA disabled) or a short-lived MFA challenge token. The frontend stores the challenge, shows a 6-digit code screen, then exchanges the challenge + TOTP code for a real access token. Users manage TOTP enrollment (enable/disable) from /setup.

**Tech Stack:** `pyotp>=2.9.0` (TOTP), `qrcode.react` (frontend QR rendering), FastAPI JWT, React 19 + TanStack Query

## Task 4: Backend — MFA challenge token helpers in auth.py

**Files:**
- Modify: `backend/app/auth.py`

- [ ] **Step 1: Add MFA token functions to auth.py**

  Open `backend/app/auth.py`. After the `create_access_token` function, add:

  ```python
  MFA_CHALLENGE_EXPIRE_MINUTES = 5   # short window — user must enter code within 5 min


  def create_mfa_challenge_token(username: str) -> str:
      """Create a short-lived JWT that represents a pending MFA challenge.

      This token is NOT an access token — it only grants the right to call
      POST /api/auth/mfa/verify. The frontend stores it temporarily between
      login steps. It carries type='mfa_challenge' so get_current_user rejects it.
      """
      payload = {
          "sub": username,
          "type": "mfa_challenge",
          "exp": datetime.now(timezone.utc) + timedelta(minutes=MFA_CHALLENGE_EXPIRE_MINUTES),
      }
      return jwt.encode(payload, SECRET_KEY, algorithm=ALGORITHM)


  def verify_mfa_challenge_token(token: str) -> str:
      """Validate an MFA challenge token and return the username.

      Raises HTTPException 401 if invalid, expired, or wrong type.
      """
      exc = HTTPException(
          status_code=status.HTTP_401_UNAUTHORIZED,
          detail="Invalid or expired MFA session",
      )
      try:
          payload = jwt.decode(token, SECRET_KEY, algorithms=[ALGORITHM])
          if payload.get("type") != "mfa_challenge":
              raise exc
          username: str = payload.get("sub", "")
          if not username:
              raise exc
          return username
      except JWTError:
          raise exc
  ```

## Task 6: Backend — MFA router (status, setup, enable, disable, verify)

**Files:**
- Create: `backend/app/routers/mfa.py`

- [ ] **Step 1: Create mfa.py router**

  Create `backend/app/routers/mfa.py` with this full content:

  ```python
  import pyotp
  from typing import Annotated
  from fastapi import APIRouter, Depends, HTTPException
  from sqlalchemy.orm import Session
  from .. import models, schemas
  from ..database import get_db
  from ..auth import get_current_user, create_access_token, verify_mfa_challenge_token

  router = APIRouter(prefix="/api/auth/mfa", tags=["mfa"])

  ISSUER = "ServerInventory"


  @router.get("/status", response_model=schemas.MfaStatusResponse)
  def mfa_status(
      current_user: Annotated[models.User, Depends(get_current_user)],
  ) -> schemas.MfaStatusResponse:
      return schemas.MfaStatusResponse(enabled=bool(current_user.totp_enabled))


  @router.post("/setup", response_model=schemas.MfaSetupResponse)
  def mfa_setup(
      current_user: Annotated[models.User, Depends(get_current_user)],
  ) -> schemas.MfaSetupResponse:
      """Generate a fresh TOTP secret and provisioning URI.

      The secret is NOT saved yet — the user must call /enable with a valid
      code to prove they've enrolled the secret in their authenticator app.
      """
      secret = pyotp.random_base32()
      totp = pyotp.TOTP(secret)
      uri = totp.provisioning_uri(name=current_user.username, issuer_name=ISSUER)
      return schemas.MfaSetupResponse(secret=secret, uri=uri)


  @router.post("/enable", status_code=204)
  def mfa_enable(
      payload: schemas.MfaEnableRequest,
      current_user: Annotated[models.User, Depends(get_current_user)],
      db: Annotated[Session, Depends(get_db)],
  ) -> None:
      """Save the TOTP secret and enable MFA.

      Requires the user to supply the current code from their authenticator
      to prove they have successfully enrolled before we store and activate.
      """
      if current_user.totp_enabled:
          raise HTTPException(status_code=400, detail="MFA is already enabled")
      totp = pyotp.TOTP(payload.secret)
      if not totp.verify(payload.code, valid_window=1):
          raise HTTPException(status_code=400, detail="Invalid TOTP code")
      current_user.totp_secret = payload.secret
      current_user.totp_enabled = True
      db.commit()


  @router.post("/disable", status_code=204)
  def mfa_disable(
      payload: schemas.MfaDisableRequest,
      current_user: Annotated[models.User, Depends(get_current_user)],
      db: Annotated[Session, Depends(get_db)],
  ) -> None:
      """Disable MFA after verifying the current TOTP code."""
      if not current_user.totp_enabled or not current_user.totp_secret:
          raise HTTPException(status_code=400, detail="MFA is not enabled")
      totp = pyotp.TOTP(current_user.totp_secret)
      if not totp.verify(payload.code, valid_window=1):
          raise HTTPException(status_code=400, detail="Invalid TOTP code")
      current_user.totp_secret = None
      current_user.totp_enabled = False
      db.commit()


  @router.post("/verify", response_model=schemas.LoginResponse)
  def mfa_verify(
      payload: schemas.MfaVerifyRequest,
      db: Annotated[Session, Depends(get_db)],
  ) -> schemas.LoginResponse:
      """Exchange an MFA challenge token + TOTP code for a full access token."""
      username = verify_mfa_challenge_token(payload.mfa_token)
      user = db.query(models.User).filter(models.User.username == username).first()
      if not user or not user.is_active:
          raise HTTPException(status_code=401, detail="User not found or inactive")
      if not user.totp_enabled or not user.totp_secret:
          raise HTTPException(status_code=400, detail="MFA is not configured for this user")
      totp = pyotp.TOTP(user.totp_secret)
      if not totp.verify(payload.code, valid_window=1):
          raise HTTPException(status_code=400, detail="Invalid TOTP code")
      token = create_access_token({"sub": user.username, "role": user.role})
      return schemas.LoginResponse(
          access_token=token,
          token_type="bearer",
          role=user.role,
          username=user.username,
      )
  ```

## Task 9: Frontend — Update useAuth.ts for MFA challenge flow

**Files:**
- Modify: `frontend/src/hooks/useAuth.ts`

The `login()` function must handle the MFA challenge case. When the backend returns `mfa_required: true`, store the challenge token in state and expose it through context. Add `completeMfa()` and `cancelMfa()` to AuthContext.

- [ ] **Step 1: Rewrite useAuth.ts**

  Replace the entire content of `frontend/src/hooks/useAuth.ts` with:

  ```typescript
  import {
    createContext,
    createElement,
    useCallback,
    useContext,
    useEffect,
    useState,
    type ReactNode,
  } from 'react'
  import { http, mfaApi } from '../api'

  export type UserRole = 'admin' | 'write' | 'read'

  export interface AuthUser {
    username: string
    role: UserRole
  }

  export interface MfaChallenge {
    mfa_token: string
    username: string
  }

  interface AuthContextValue {
    user: AuthUser | null
    mfaChallenge: MfaChallenge | null
    login: (username: string, password: string, rememberMe?: boolean) => Promise<void>
    completeMfa: (code: string) => Promise<void>
    cancelMfa: () => void
    logout: () => void
  }

  export const AuthContext = createContext<AuthContextValue | null>(null)

  function loadStored(): { user: AuthUser | null; token: string | null } {
    try {
      const token = localStorage.getItem('si_token')
      const raw = localStorage.getItem('si_user')
      const user = raw ? (JSON.parse(raw) as AuthUser) : null
      return { token, user }
    } catch {
      return { token: null, user: null }
    }
  }

  function setAxiosToken(token: string | null) {
    if (token) {
      http.defaults.headers.common['Authorization'] = `Bearer ${token}`
    } else {
      delete http.defaults.headers.common['Authorization']
    }
  }

  export function AuthProvider({ children }: { children: ReactNode }) {
    const stored = loadStored()
    const [user, setUser] = useState<AuthUser | null>(stored.user)
    const [mfaChallenge, setMfaChallenge] = useState<MfaChallenge | null>(null)

    useEffect(() => {
      setAxiosToken(stored.token)
    }, [])  // eslint-disable-line react-hooks/exhaustive-deps

    useEffect(() => {
      const handler = () => { setUser(null) }
      window.addEventListener('auth:expired', handler)
      return () => window.removeEventListener('auth:expired', handler)
    }, [])

    const login = useCallback(async (username: string, password: string, rememberMe = false) => {
      const params = new URLSearchParams({ username, password })
      params.append('remember_me', rememberMe ? 'true' : 'false')
      const res = await http.post('/api/auth/login', params, {
        headers: { 'Content-Type': 'application/x-www-form-urlencoded' },
      })
      const data = res.data as {
        access_token?: string
        role?: string
        mfa_required?: boolean
        mfa_token?: string
      }

      if (data.mfa_required && data.mfa_token) {
        // Password OK, MFA required — store challenge, wait for TOTP code.
        setMfaChallenge({ mfa_token: data.mfa_token, username })
        return
      }

      // Normal login — store token and user.
      const { access_token, role } = data as { access_token: string; role: UserRole }
      const authUser: AuthUser = { username, role }
      localStorage.setItem('si_token', access_token)
      localStorage.setItem('si_user', JSON.stringify(authUser))
      setAxiosToken(access_token)
      setUser(authUser)
    }, [])

    const completeMfa = useCallback(async (code: string) => {
      if (!mfaChallenge) throw new Error('No MFA challenge in progress')
      const res = await mfaApi.verify(mfaChallenge.mfa_token, code)
      const { access_token, role } = res.data as { access_token: string; role: UserRole }
      const authUser: AuthUser = { username: mfaChallenge.username, role }
      localStorage.setItem('si_token', access_token)
      localStorage.setItem('si_user', JSON.stringify(authUser))
      setAxiosToken(access_token)
      setMfaChallenge(null)
      setUser(authUser)
    }, [mfaChallenge])

    const cancelMfa = useCallback(() => {
      setMfaChallenge(null)
    }, [])

    const logout = useCallback(() => {
      localStorage.removeItem('si_token')
      localStorage.removeItem('si_user')
      setAxiosToken(null)
      setUser(null)
      setMfaChallenge(null)
    }, [])

    return createElement(
      AuthContext.Provider,
      { value: { user, mfaChallenge, login, completeMfa, cancelMfa, logout } },
      children,
    )
  }

  export function useAuth(): AuthContextValue {
    const ctx = useContext(AuthContext)
    if (!ctx) throw new Error('useAuth must be inside <AuthProvider>')
    return ctx
  }
  ```

            <Text>Enter the 6-digit code from your authenticator app to continue.</Text>

            <form onSubmit={handleSubmit}>
              <Flex direction="column" gap={4}>
                <Input
                  type="text"
                  inputMode="numeric"
                  pattern="\d{6}"
                  maxLength={6}
                  placeholder="000000"
                  value={code}
                  onChange={e => setCode(e.target.value)}
                  autoFocus
                  css={{ fontSize: '$xl', letterSpacing: '0.3em', textAlign: 'center' }}
                />

                {error && (
                  <Text css={{ color: '$error', fontSize: '$sm' }}>{error}</Text>
                )}

                <Button type="submit" disabled={loading} css={{ width: '100%' }}>
                  {loading ? 'Verifying…' : 'Verify'}
                </Button>
              </Flex>
            </form>

            <Button
              variant="ghost"
              onClick={cancelMfa}
              css={{ alignSelf: 'flex-start', padding: 0, color: '$textMuted' }}
            >
              <ArrowLeft size={14} style={{ marginRight: 4 }} />
              Back to login
            </Button>
          </Flex>
        </Card>
      </Flex>
    )
  }

## Task 12: Frontend — MfaSetupSection component

**Files:**
- Create: `frontend/src/components/MfaSetupSection.tsx`

This component is mounted inside the /setup page. It shows current MFA status and lets the user enable or disable TOTP.

- [ ] **Step 1: Create MfaSetupSection.tsx**

  Create `frontend/src/components/MfaSetupSection.tsx`:

  ```tsx
  import { useState } from 'react'
  import { useQuery, useMutation, useQueryClient } from '@tanstack/react-query'
  import QRCode from 'qrcode.react'
  import { Shield, ShieldOff, ShieldCheck } from 'lucide-react'
  import { mfaApi } from '../api'
  import { useToast } from '../hooks/useToast'
  import { Card, Flex, Heading, Text, Input, Button, Badge } from './StitchUI'

  type Step = 'idle' | 'setup' | 'disable'

  interface SetupData {
    secret: string
    uri: string
  }

  export default function MfaSetupSection() {
    const { toast } = useToast()
    const qc = useQueryClient()
    const [step, setStep] = useState<Step>('idle')
    const [setupData, setSetupData] = useState<SetupData | null>(null)
    const [code, setCode] = useState('')
    const [fieldError, setFieldError] = useState('')

    const { data: statusData } = useQuery({
      queryKey: ['mfa-status'],
      queryFn: () => mfaApi.status().then(r => r.data),
    })

    const enabled = statusData?.enabled ?? false

    const setupMutation = useMutation({
      mutationFn: () => mfaApi.setup().then(r => r.data),
      onSuccess: (data) => {
        setSetupData(data)
        setStep('setup')
        setCode('')
        setFieldError('')
      },
    })

    const enableMutation = useMutation({
      mutationFn: ({ secret, code }: { secret: string; code: string }) =>
        mfaApi.enable(secret, code),
      onSuccess: () => {
        toast.success('Two-factor authentication enabled')
        qc.invalidateQueries({ queryKey: ['mfa-status'] })
        setStep('idle')
        setSetupData(null)
        setCode('')
        setFieldError('')
      },
      onError: (e: unknown) => {
        const detail = (e as { response?: { data?: { detail?: string } } })?.response?.data?.detail
        setFieldError(detail ?? 'Failed to enable MFA')
      },
    })

    const disableMutation = useMutation({
      mutationFn: (code: string) => mfaApi.disable(code),
      onSuccess: () => {
        toast.success('Two-factor authentication disabled')
        qc.invalidateQueries({ queryKey: ['mfa-status'] })
        setStep('idle')
        setCode('')
        setFieldError('')
      },
      onError: (e: unknown) => {
        const detail = (e as { response?: { data?: { detail?: string } } })?.response?.data?.detail
        setFieldError(detail ?? 'Invalid code')
      },
    })

    function startEnable() {
      setupMutation.mutate()
    }

    function confirmEnable(e: React.FormEvent) {
      e.preventDefault()
      setFieldError('')
      if (!setupData) return
      const trimmed = code.replace(/\s/g, '')
      if (!/^\d{6}$/.test(trimmed)) {
        setFieldError('Enter the 6-digit code from your authenticator app')
        return
      }
      enableMutation.mutate({ secret: setupData.secret, code: trimmed })
    }

    function confirmDisable(e: React.FormEvent) {
      e.preventDefault()
      setFieldError('')
      const trimmed = code.replace(/\s/g, '')
      if (!/^\d{6}$/.test(trimmed)) {
        setFieldError('Enter the 6-digit code from your authenticator app')
        return
      }
      disableMutation.mutate(trimmed)
    }

    function cancel() {
      setStep('idle')
      setSetupData(null)
      setCode('')
      setFieldError('')
    }

    return (
      <Card css={{ padding: '$5' }}>
        <Flex direction="column" gap={4}>
          <Flex justify="between" align="center">
            <Flex align="center" gap={2}>
              {enabled ? <ShieldCheck size={20} /> : <Shield size={20} />}
              <Heading as="h3" css={{ margin: 0 }}>Two-Factor Authentication</Heading>
            </Flex>
            <Badge variant={enabled ? 'success' : 'default'}>
              {enabled ? 'Enabled' : 'Disabled'}
            </Badge>
          </Flex>

          <Text variant="muted">
            {enabled
              ? 'Your account is protected with TOTP two-factor authentication.'
              : 'Add an extra layer of security using an authenticator app (Google Authenticator, Authy, etc.).'}
          </Text>

          {/* Idle state */}
          {step === 'idle' && (
            <Flex gap={3}>
              {enabled ? (
                <Button
                  variant="danger"
                  onClick={() => { setStep('disable'); setCode(''); setFieldError('') }}
                >
                  <ShieldOff size={14} style={{ marginRight: 6 }} />
                  Disable 2FA
                </Button>
              ) : (
                <Button onClick={startEnable} disabled={setupMutation.isPending}>
                  <Shield size={14} style={{ marginRight: 6 }} />
                  {setupMutation.isPending ? 'Generating…' : 'Enable 2FA'}
                </Button>
              )}
            </Flex>
          )}

          {/* Setup flow: show QR + code entry */}
          {step === 'setup' && setupData && (
            <form onSubmit={confirmEnable}>
              <Flex direction="column" gap={4}>
                <Text>
                  Scan this QR code with your authenticator app, then enter the 6-digit code to confirm.
                </Text>

                <Flex justify="center">
                  <div style={{ background: 'white', padding: 12, borderRadius: 8, display: 'inline-block' }}>
                    <QRCode value={setupData.uri} size={180} level="M" />
                  </div>
                </Flex>

                <details>
                  <summary style={{ cursor: 'pointer', fontSize: '0.85rem', color: 'var(--colors-textMuted)' }}>
                    Can't scan? Enter the key manually
                  </summary>
                  <Text
                    css={{
                      fontFamily: 'monospace',
                      fontSize: '$sm',
                      background: '$surface',
                      padding: '$2 $3',
                      borderRadius: '$sm',
                      marginTop: '$2',
                      wordBreak: 'break-all',
                    }}
                  >
                    {setupData.secret}
                  </Text>
                </details>

                <Input
                  type="text"
                  inputMode="numeric"
                  pattern="\d{6}"
                  maxLength={6}
                  placeholder="000000"
                  value={code}
                  onChange={e => setCode(e.target.value)}
                  autoFocus
                  css={{ letterSpacing: '0.3em', textAlign: 'center' }}
                />

                {fieldError && (
                  <Text css={{ color: '$error', fontSize: '$sm' }}>{fieldError}</Text>
                )}

                <Flex gap={3}>
                  <Button type="submit" disabled={enableMutation.isPending}>
                    {enableMutation.isPending ? 'Enabling…' : 'Confirm & Enable'}
                  </Button>
                  <Button type="button" variant="ghost" onClick={cancel}>Cancel</Button>
                </Flex>
              </Flex>
            </form>
          )}

          {/* Disable flow: verify current code */}
          {step === 'disable' && (
            <form onSubmit={confirmDisable}>
              <Flex direction="column" gap={4}>
                <Text>Enter your current TOTP code to confirm you want to disable 2FA.</Text>

                <Input
                  type="text"
                  inputMode="numeric"
                  pattern="\d{6}"
                  maxLength={6}
                  placeholder="000000"
                  value={code}
                  onChange={e => setCode(e.target.value)}
                  autoFocus
                  css={{ letterSpacing: '0.3em', textAlign: 'center' }}
                />

                {fieldError && (
                  <Text css={{ color: '$error', fontSize: '$sm' }}>{fieldError}</Text>
                )}

                <Flex gap={3}>
                  <Button type="submit" variant="danger" disabled={disableMutation.isPending}>
                    {disableMutation.isPending ? 'Disabling…' : 'Disable 2FA'}
                  </Button>
                  <Button type="button" variant="ghost" onClick={cancel}>Cancel</Button>
                </Flex>
              </Flex>
            </form>
          )}
        </Flex>
      </Card>
    )
  }
  ```
